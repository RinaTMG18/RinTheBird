Jag har skapat ett Flappy Bird liknande spel där du styr en fågel som ska flyga genom rör och samla poäng.
Spelets funktioner: Endless bakgrund: Moln och byggnader som rör sig kontinuerligt.

Poängsystem: Poäng varje gång fågeln passerar ett rör.
Hinder: Vertikala rör med varierande avstånd.
Kontroller: Tryck för att få fågeln att flyga uppåt; gravitation drar den neråt. 
Restart: Spelet kan startas om när man trycker på "Play Again"
